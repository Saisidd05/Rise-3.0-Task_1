# Screenshots Directory

This directory contains screenshots and videos demonstrating the Currency Converter GUI application.

## Contents

- Screenshots: PNG/JPG images of the application interface
- Videos: MP4/AVI files showing the application in action

## How to Add

1. Run the application on your local machine
2. Take screenshots of different views (main interface, conversion results, etc.)
3. Record short videos demonstrating the conversion process
4. Save files here with descriptive names
5. Commit and push to the repository
